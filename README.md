# ihcux-lista-01

Este repositório possui um programa simples em C# que simula um sistema console de uma cantina universitária. Nele foram implementadas 3 heurísticas, sendo elas:

1. Heurística #1 (Visibilidade do Status): Mostra uma mensagem simples que indica em qual etapa do pedido o aluno está (ex: [Passo 1 de 3] Seleção de Item).
2. Heurística #3 (Controle e Liberdade): O aluno consegue voltar em qualquer etapa para corrigir a informação anterior, ou cancelar para abortar tudo.
3. Heurística #9 (Ajuda e Erros): Se o aluno digitar um código de produto que não existe, o sistema não pode apenas exibir uma mensagem genérica. Ele deve diz que o código não foi encontrado.
